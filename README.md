# Odin Recipes

Proyek ini dibuat sebagai bagian dari pembelajaran **The Odin Project** untuk memahami dasar-dasar HTML, struktur halaman web, dan penggunaan link antar halaman. Project ini berisi kumpulan resep sederhana yang ditulis menggunakan HTML.

## Tujuan
- Melatih penggunaan elemen dasar HTML.
- Membuat struktur website sederhana dengan beberapa halaman.
- Belajar menggunakan link (`<a>`) untuk navigasi antar halaman.
- Memahami penggunaan list, heading, paragraf, dan gambar.
